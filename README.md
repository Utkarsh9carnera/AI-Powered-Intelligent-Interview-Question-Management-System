# AI-Powered Intelligent Interview Question Management System

## Overview
This project is an AI-powered interview question management platform that enables users to search, manage, and study interview questions using AI-assisted search. The system supports role-based access control with four user roles: Root Admin, Admin, Question Manager, and Viewer.

## Features
- Google OAuth Authentication
- Role-Based Access Control (RBAC)
- AI-Powered Interview Question Search
- Question Management
- Metadata Management
- Search History
- Responsive User Interface

## Technology Stack

### Frontend
- React.js

### Backend
- ASP.NET Core Web API

### Database
- PostgreSQL

### AI Integration
- Groq API
